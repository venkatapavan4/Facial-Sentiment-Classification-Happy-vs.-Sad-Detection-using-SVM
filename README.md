# Facial Sentiment Classification: Happy vs Sad (SVM)

Designed a lightweight facial emotion detection system that leverages
handcrafted HOG features to capture subtle facial patterns, enabling
efficient Happy vs Sad classification without deep learning.\
Emphasized model interpretability and performance trade-offs by
benchmarking multiple ML algorithms and validating robustness through
cross-validation and visual error analysis.

## Tech Stack

-   Python
-   OpenCV
-   scikit-image
-   scikit-learn
-   NumPy, Pandas
-   Matplotlib

## Key Highlights

-   Achieved \~89% accuracy
-   Compared SVM, Random Forest, and KNN
-   Used HOG feature extraction
-   Visualized predictions and confusion matrix
